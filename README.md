# 🍱 自炊シミュレーター

一人暮らしを始める人向けの食事費シミュレーター。外食・弁当・自炊の1ヶ月の食費を比較し、自炊で本当に節約できるのかが一目でわかります。

## 特徴

- **30日間のシミュレーション** — カレンダー形式で毎日の食事を選択
- **リアルタイム集計** — 合計金額・1日平均・節約額をリアルタイムで表示
- **複数の食事選択肢** — 外食・弁当・自炊それぞれで複数のメニューを用意
- **データ分析** — 食事パターンを可視化、支出額とカテゴリ別分析
- **CSV エクスポート** — 結果をダウンロード可能

## デモ

[🌐 https://flipslidersand.github.io/food-simulator](https://flipslidersand.github.io/food-simulator)

## 使い方

1. カレンダーの各日付をクリック
2. 外食・弁当・自炊から食事を選択
3. 1ヶ月分入力すると、統計が自動計算されます
4. 「CSV にエクスポート」で結果をダウンロード可能

## 開発

### セットアップ

```bash
git clone https://github.com/flipslidersand/food-simulator.git
cd food-simulator
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開く

### ビルド

```bash
npm run build
```

`dist/` に本番用ファイルが生成されます。
